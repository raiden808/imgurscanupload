# ImgurOCR Scanner

## Scan an image instantly!
An app using [Tesseract.js](https://tesseract.projectnaptha.com/) and Imgur API to scan your image for text.

### Notes:
- **Imgur API** - If you want to fork be sure to add you own key on an .env file.
